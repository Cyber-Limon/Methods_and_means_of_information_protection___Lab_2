# Методы и средства защиты информации
# Лабораторная работа 2

### Цель работы
Целью лабораторной работы является освоение принципа работы алгоритма "AES" и приобретение практических навыков шифрования данных при помощи данного алгоритма в программном пакете "CrypTool 1.4.30".

### Задания
1. Познакомиться с операциями алгоритма и преобразованиями, производимыми в этих операциях с помощью анимации (Indiv.Procedures – Visualization of Algorithms – AES – Rijndael Animation).
2. Зашифровать данные с помощью ключа, попробовать расшифровать с помощью правильного и неправильного ключа, посмотреть, как меняются входные данные по раундам. Данные и ключ можно вводить в пределах 0-1 и a-d (Indiv.Procedures – Visualization of Algorithms – AES – Rijndael Inspector).
3. Создать документ (File – New) с произвольной информацией и зашифровать его с помощью ключа (Encrypt/Decrypt – Symmetric (modern) – Rijndael (AES)), посмотреть, как выглядит зашифрованная информация в шестнадцатеричной системе счисления и в виде текста. Сохранить документ в зашифрованном виде, удалив при этом исходный.
4. Провести анализ зашифрованного документа, посмотреть параметры анализа алгоритма, оценить, сколько времени займет анализ и результаты анализа (Analysis – Symmetric Encryption (modern) – Rijndael (AES)). Попробовать провести анализ алгоритма с помощью ключа другой длины, с помощью частично известного ключа и оценить результаты анализа.
5. Попробовать расшифровать сохраненный документ с помощью правильного и неправильного ключа, с помощью ключа другой длины (Encrypt/Decrypt – Symmetric (modern) – Rijndael (AES)).
6. Создать новый документ и зашифровать его с помощью простого ключа (например, 10 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00). Провести анализ данного документа, оценить результаты анализа. Сделать выводы о проделанной работе.
